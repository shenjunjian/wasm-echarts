wasm-echarts 文档

在浏览器侧对齐 ECharts canvas 模式的核心用法。

快速开始

cd crates/wasm-echarts && wasm-pack build --target web

cd ../..
cd site && npm install && npm run dev
# 打开 http://127.0.0.1:5173/echarts/examples/

JS 薄壳 API

import echarts from './js/echarts.js';

const chart = await echarts.init(dom, { renderer: 'canvas' });
chart.setOption(option);
chart.on('click', ({ hit }) => { /* ... */ });
chart.resize();
chart.benchmark(30);
chart.dispose();

函数 option

WASM 侧可识别 option 中的函数字段（如 tooltip.formatter、series[].itemStyle.color）， 由 JS 薄壳在 setOption 时保留并在渲染/交互阶段回调。实例页目前以 JSON 为主；含函数的 option 见 交互合集 等页面内置逻辑。

已支持图表与交互（MVP）

  • 图表：line、bar、pie、scatter
  • 交互：hover 高亮、toggleSelect、wheel dataZoom、axisPointer 竖线、string tooltip
  • 轴标签 Text 渲染（formatter 解析存在，轴侧调用待完善）

与官方 ECharts 差异

  • 无 SVG、无动画过渡、legend / dataZoom 滑块未绘制
  • series label、RichText、gauge、polar 等待实现
  • tooltip 暂不支持 HTMLElement 返回值
