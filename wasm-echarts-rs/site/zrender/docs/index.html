wasm-zrender 文档

底层 2D 渲染 WASM 绑定，直接封装 rust_zrender::ZRenderer。

快速开始

cd crates/wasm-zrender && wasm-pack build --target web

cd ../..
cd site && npm install && npm run dev
# 打开 http://127.0.0.1:5173/zrender/examples/

主要 API

  • new ZRenderInstance(width, height, dpr)
  • load_scene(scene) — 内置场景：shapes | text | sector | hit | state
  • refresh() → RGBA 字节数组
  • find_hover(x, y) → pathIndex / seriesIndex / dataIndex
  • highlight_path / downplay_path
  • resize(width, height, dpr)

架构关系

wasm-zrender 与 wasm-echarts 为 sibling，均只依赖 rust-zrender。 echarts 图表管线不经过 wasm-zrender，而是内部直接使用 ZRenderer。

已知限制

  • 当前仅 load_scene 预设场景，动态 add_path API 待补
  • Text 未完全纳入场景图 z 排序；conic 渐变未实现
  • 输出为 RGBA 像素，需 JS 侧 putImageData 上屏
